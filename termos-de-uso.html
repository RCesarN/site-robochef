2. Descrição do Serviço

O RoboChef Publicador é um script de automação local que utiliza a TikTok Content Posting API para publicar vídeos de receitas culinárias produzidos pelo criador na plataforma TikTok. O fluxo de integração inclui:

  • Autenticação via OAuth 2.0 com o Login Kit do TikTok
  • Upload de vídeos usando a Content Posting API (video.upload)
  • Publicação direta ou via rascunho (video.publish)

3. Uso Permitido

O Aplicativo destina-se exclusivamente a:

  • Automatizar a publicação de vídeos de receitas na conta TikTok do próprio desenvolvedor
  • Gerenciar o agendamento de publicações de conteúdo culinário original
  • Uso estritamente pessoal, sem geração de receita direta pela ferramenta

4. Restrições de Uso
