Política de Privacidade

RoboChef Publicador  ·  Última atualização: Junho de 2026

1. Introdução

Esta Política de Privacidade descreve como o RoboChef Publicador trata as informações no contexto de sua integração com a API do TikTok. O Aplicativo é uma ferramenta de automação pessoal, sem fins comerciais, usada exclusivamente pelo criador da conta @robochefbrasil.

O RoboChef Publicador não coleta, armazena nem compartilha dados pessoais de terceiros. Todos os dados processados pertencem exclusivamente ao próprio desenvolvedor/usuário da ferramenta.

2. Dados Coletados

O Aplicativo processa apenas os seguintes dados mínimos necessários para seu funcionamento:

Dado Finalidade Armazenamento
Access Token OAuth2 Autenticação com a API do TikTok Local (arquivo JSON protegido)
Refresh Token OAuth2 Renovação automática do acesso Local (arquivo JSON protegido)
Client Key / Secret Identificação do aplicativo na API Local (arquivo JSON protegido)

Esses dados são armazenados exclusivamente no dispositivo local do desenvolvedor e nunca são transmitidos a servidores de terceiros além da API oficial do TikTok.

3. Dados do TikTok Acessados

Por meio dos escopos autorizados via OAuth2, o Aplicativo acessa:

  • user.info.basic: Nome de exibição e avatar público da conta (para validação de login)
  • video.upload: Permissão para enviar vídeos como rascunho para a conta do criador
  • video.publish: Permissão para publicar vídeos diretamente no perfil do criador

Nenhuma dessas informações é compartilhada com terceiros ou utilizada para qualquer finalidade além da publicação automatizada de conteúdo.

4. Compartilhamento de Dados

O RoboChef Publicador não compartilha nenhum dado com terceiros. As únicas comunicações de rede realizadas pelo Aplicativo são:

  • Requisições para os endpoints oficiais da TikTok API (open.tiktokapis.com)
  • Requisições de autenticação para www.tiktok.com/v2/auth/authorize/

5. Retenção de Dados

Os tokens de autenticação são mantidos localmente enquanto forem necessários para o funcionamento do Aplicativo. O usuário pode revogar o acesso a qualquer momento diretamente nas configurações de privacidade da sua conta TikTok (Configurações → Segurança → Aplicativos autorizados).

6. Segurança

Os arquivos de credenciais são armazenados localmente no dispositivo do desenvolvedor, com acesso restrito ao usuário do sistema operacional. O Aplicativo não transmite credenciais por redes não seguras.

7. Direitos do Usuário (LGPD)

Em conformidade com a Lei Geral de Proteção de Dados (Lei nº 13.709/2018), o usuário tem direito a:

  • Acessar os dados armazenados localmente a qualquer momento
  • Solicitar a exclusão dos tokens de autenticação
  • Revogar o acesso do Aplicativo à conta TikTok a qualquer momento

8. Cookies e Rastreamento

O RoboChef Publicador não utiliza cookies e não realiza nenhuma forma de rastreamento de usuários. Este site (GitHub Pages) tampouco coleta dados de visitantes.

9. Alterações nesta Política

Esta Política pode ser atualizada periodicamente. Alterações significativas serão refletidas na data de "Última atualização" no topo desta página.

10. Contato

Para exercer seus direitos ou esclarecer dúvidas sobre esta Política de Privacidade, entre em contato pelo perfil do TikTok: @robochefbrasil.
